Alphagetti Pasta

Package Redesign (2022)

Description

This project was a redesign for Alphagetti pasta. We had to redesign the package and the logo. We had to create a label and make sure that the elements present on the label are readable, clear, and capture the playfulness of the product. I created a mascot for the product, a fun and smiley "A" to embody the energy of Alphagetti. We used a real can and modeled our label to better envision what the end result of the product would look like and the establish the right proportions.

Details

This project was created using Illustrator, InDesign, and Photoshop. This label was printed and tested on a real can. These illustrations were created from cratched based off of the sketches found below.
